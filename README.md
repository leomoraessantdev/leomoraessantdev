Comecei a estudar programação em **2022**, tendo o **Python** como primeira linguagem. De lá pra cá migrei para o desenvolvimento web, construindo aplicações full-stack com **React, TypeScript e Node.js**. Já publiquei projetos pessoais em produção — de ferramentas com IA a SPAs completas — cuidando de tudo: do design da interface ao deploy. Sempre em busca de aprender mais e encarar novos desafios.
